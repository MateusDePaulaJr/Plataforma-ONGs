# Plataforma de ONGs – Entrega Final

Projeto desenvolvido para a disciplina de Desenvolvimento Front-End para Web.

---

## ✔️ Funcionalidades Implementadas

### **Acessibilidade (WCAG – mínimo obrigatório)**
- Link “Pular para o conteúdo” no início da página
- Conteúdo principal marcado com `<main id="conteudo">`
- Labels conectados a inputs em formulários
- Modo claro e escuro ativado por uma *chavinha* acessível
- Estrutura semântica (`header`, `nav`, `main`, `footer`)

### **Tema (Claro / Escuro)**
- Chave no topo da página
- Alternância imediata entre os temas
- Implementado com `data-theme` no elemento `<html>`  
- CSS simples e direto, sem dependências externas

---

## ✔️ GitFlow adotado

Branchs utilizadas:
- `main` – versão final e estável
- `develop` – integração de features antes da release
- `feature/acessibilidade` – implementação da entrega

### Commits semânticos
Exemplos:
- `feat(tema): adicionar chavinha claro/escuro`
- `fix(acessibilidade): ajustar skip link`
- `chore(build): preparar arquivos`

### Pull Requests
- feature/acessibilidade → develop  
- develop → main (release final)
